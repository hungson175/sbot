# sbot Learning Modules

Hands-on lessons for the technologies behind sbot. Each module teaches by building.

## Modules

| # | Module | Topics | Time |
|---|--------|--------|------|
| 01 | [Concurrency Programming in Python](01_concurrency_python/) | async/await, event loop, Queue, create_task, buffering bugs, Redis Streams | ~2h |
| 02 | [Understanding MCP by Building It](02_mcp_protocol/) | JSON-RPC 2.0, stdio transport, MCP server, MCP client, FastMCP SDK | ~1.5h |

## How to Use

```bash
cd learn/01_concurrency_python/   # or 02_mcp_protocol/
jupyter notebook                  # open in browser, run cells, fill in answers
```

Each notebook has:
- Explanation of the concept
- Code exercises with TODOs for you to fill in
- Test cells that verify your implementation
- Questions to check understanding
