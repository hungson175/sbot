Search for a text pattern across files in a directory tree using ripgrep. Returns matching lines with file paths and line numbers. Use this BEFORE editing to find where things are defined, where functions are called, or where variables are used.

Args:
    pattern: Regex pattern to search for (e.g. "def main", "class Config", "import os", "TODO").
    path: Directory to search in (default: current directory ".", e.g. "src/", "tests/").
    include: Glob pattern to filter files (e.g. "*.py", "*.ts", "*.md"). If empty, searches all text files.
    max_results: Maximum number of matching lines to return (default 50, increase for broader searches).
